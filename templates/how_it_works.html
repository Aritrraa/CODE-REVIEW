Step-by-Step Process

  1. File Upload: The user uploads a Python file (.py) for analysis.
  2. Static Analysis with Pylint: The app runs Pylint to count errors (E), warnings (W), and conventions (C).
  3. Machine Learning Inference:
    • CodeBERT: The code is tokenized and passed to a fine-tuned CodeBERT model which classifies the code as Good or Needs Improvement.
    • Logistic Regression: Based on (E, W, C) counts, the logistic model also predicts the same label independently.
  4. Result Display: The result page shows both model predictions, a confidence score, and line-specific issues.

Additional Features

  • Line-level message highlighting
  • Syntax highlighting for better readability
  • Option to download or view code inline
